## 📍 經緯度座標說明

### 資料結構
```typescript
{
  longitude: number  // 經度（必填）
  latitude: number   // 緯度（必填）
  height?: number    // 高度（可選，單位：米）
}
```

### 當前假資料座標範圍
- **經度範圍**: 121.522 ~ 121.525
- **緯度範圍**: 25.101 ~ 25.103
- **中心點**: 121.523°E, 25.102°N（芝山岩）

### 如何更新為真實座標

1. **查找真實景點座標**
   - 使用 Google Maps 右鍵點擊位置 → 複製座標
   - 或使用其他地圖服務查詢

2. **更新資料檔案**
   編輯 `src/data/sites.ts`，修改對應的 `longitude` 和 `latitude`：

   ```typescript
   {
     id: '1',
     name: '芝山岩遺址',
     longitude: 121.523456,  // 更新為真實經度
     latitude: 25.102345,     // 更新為真實緯度
     // ... 其他屬性
   }
   ```

3. **座標格式**
   - 經度：121.xxx（東經，正值）
   - 緯度：25.xxx（北緯，正值）
   - 使用小數點格式（十進制度數）
